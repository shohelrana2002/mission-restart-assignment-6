# SwiftCart - All Products Page


## JavaScript Questions ans:

### 1 Null vs Undefined
- `null` → মান  খালি রাখা 
- `undefined` → মান declare করা হয়েছে কিন্তু assign করা হয়নি

### 2 Map() vs forEach() difarnce
- `map()` → নতুন array return করে 
- `forEach()` → কোনো return value দেয় না, শুধু iterate করে

### 3 == vs === diffrance ki ?
- `==` → value check করে, type convert করে
- `===` → value + type দুটোই check করে।  

### 4 Async/Await ki ?
- API call বা Promise handle করার জন্য 
- কোড clean ও readable হয়


### 5 Global, Function, Block ki ?
- Global Scope → সব জায়গা থেকে accessable
- Function Scope → শুধু function এর ভিতর accessable
- Block Scope → {} এর ভিতর declare করা variable শুধু ওই block এ accessable
